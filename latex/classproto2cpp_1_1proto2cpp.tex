\subsection{Detailed Description}
Class for converting Google Protocol Buffers .proto files into C++ style output to enable Doxygen usage. 

The C++ style output is printed into standard output.~\newline
 There are three different logging levels for the class\+: 
\begin{DoxyItemize}
\item \hyperlink{classproto2cpp_1_1proto2cpp_a85f1be79a05376f69b9a75c28ebdd29c}{log\+None}\+: do not log anything 
\item \hyperlink{classproto2cpp_1_1proto2cpp_ad88c9b4f3034ecbebc9980aa0d0de616}{log\+Errors}\+: log errors only 
\item \hyperlink{classproto2cpp_1_1proto2cpp_a0608421165933b9b58e98ac02bbc6141}{log\+All}\+: log everything
\end{DoxyItemize}Logging level is determined by {\ttfamily \hyperlink{classproto2cpp_1_1proto2cpp_a56767af005e7589efa89611e979ebd34}{log\+Level}}.~\newline
 Error logs are written to file determined by {\ttfamily \hyperlink{classproto2cpp_1_1proto2cpp_a6bcbded937d299e48e0fb03a1dce9709}{error\+Log\+File}}.~\newline
 Debug logs are written to file determined by {\ttfamily \hyperlink{classproto2cpp_1_1proto2cpp_a06614bc26f0849809d3bf6b5092f8f9c}{log\+File}}.
